# mmfwebsite
ETÜ Destek Sistemi
Erzurum Teknik Üniversitesi Yapım Onarım Destek Sistemi web uygulaması.

Özellikler
Personel ve Yönetici giriş sistemi
Destek talebi oluşturma (Kalorifer, Elektrik, Mobilya, Tesisat, Fayans Boya, Metal İşleri, Zemin Yıkama, Diğer)
Talep takip sistemi
Yönetici paneli ile talep yönetimi
Fotoğraf yükleme desteği
Kullanım
index.html dosyasını bir web tarayıcısında açın
Giriş yapın:
Personel: Kullanıcı adı: personel, Şifre: 1234
Yönetici: Kullanıcı adı: admin, Şifre: 1234

HTML5
CSS3
JavaScript (Vanilla)
Font Awesome Icons
